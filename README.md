# Gemini実践研修プログラム

企業向けGemini活用研修の教材リポジトリです。2-3時間で基礎から応用まで、実務で即使えるスキルを習得できる構成になっています。

## 📖 概要

**「AIの使い方」はAIに聞けば出る。でも、実務で回している人間の血肉化された知見は、AIでは出せない。**

このプログラムは、ToB企業の実務担当者向けに、Geminiを使った業務効率化を段階的に習得できる研修教材です。単なる機能紹介ではなく、実務での運用ノウハウと組織展開のコツまでカバーしています。

## 🎯 対象者

- BtoB企業の営業、企画、マーケティング、バックオフィス担当者
- Googleワークスペースを使用している企業
- AI活用初心者から中級者まで
- 実務でのAI活用を具体的にイメージしたい方

## ✨ この研修の特徴

### 1. 実務特化型のカリキュラム

- 議事録作成、報告書作成、データ分析など実際の業務タスクで学習
- Googleワークスペース連携を最大活用（スプレッドシート、ドキュメント、ドライブ）
- 「明日から使える」を最優先にした構成

### 2. ChatGPTとの差別化を明確に

- **最大の強み：Googleドライブ直接連携**
- ファイルアップロード不要で実データを即座に分析
- 社内ナレッジの知識ベース化（NotebookLM）
- Deep Researchで社内資料を検索元に設定可能

### 3. 段階的な学習設計

基本操作（20分） → 業務文書作成（30分） → 高度な活用（30分） → ドライブ連携（30分） → ナレッジ化（30分） → 自動化（応用編）

## 📋 研修内容

### Step 1：基本操作（20分）

- チャットベースの質問
- メール文章の修正・改善

### Step 2：業務文書作成（30分）

- 議事録の自動生成
- 報告書・提案書の下書き作成

### Step 3：コンテクスト活用（30分）

- 会社情報を含めた高精度プロンプト
- Gem機能で社内コンテキストを保存
- 議事録作成専用Gemの構築

### Step 4：Googleドライブ連携（30分）

- スプレッドシートの直接分析
- Googleドキュメントの要約
- 議事録のスプレッドシート管理ワークフロー

### Step 5：NotebookLM活用（30分）

- 社内ナレッジの知識ベース化
- 顧客の声からインサイト発見
- 社内QA集・お問い合わせ対応集の構築

### Step 6：Deep Research（30分）

- 市場調査・競合分析の自動化
- 社内ドライブを検索元に設定
- 過去案件からの知見抽出

### Step 7：画像生成（10分）

- Imagen 3でプレゼン資料のビジュアル作成
- nanobananaで素早くアイコン・イラスト作成

### 💡 応用編：業務自動化

- **GAS × Gemini**：定型作業の自動化（メール→スプレッドシート自動転記など）
- **スライド自動生成**：骨組みを渡すだけで5秒でGoogleスライドが完成

## 🔥 研修後の成果
